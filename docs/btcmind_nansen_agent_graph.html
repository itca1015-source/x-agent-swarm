Agent System

Signal Source Agent

Intelligence

Job

Turn BTCMind product output and market research into structured signals.

Inputs

  • BTCMind debates and analyst disagreement
  • Wallet, flow, stablecoin, custody, bridge, and security observations
  • Manual research notes and source URLs

Output

state/btcmind_signals.json with type, summary, evidence, operator watch, risk notes.

Tell it: "Find one non-obvious crypto mechanism BTCMind can explain better than the headline. Write the signal with evidence and the condition that would make it wrong."

Poster Agent

Action

Job

Create reviewable original posts from structured signals.

Templates

  • Wallet or flow alert
  • Bull vs bear debate result
  • Hidden constraint
  • Operator watch item

Output

kind: original draft in state/reply_queue.json.

Tell it: "Write one BTCMind post that makes the reader see the machinery under the headline. No hype, no price target, no hashtag, no advice."

Continuation Agent

Action

Job

Add evidence replies under BTCMind originals, like Nansen's "Track this" and "2/" replies.

Inputs

  • Original post text
  • Evidence lines
  • BTCMind link or artifact
  • Risk and invalidation condition

Output

One to three follow-up replies that deepen trust without bloating the root post.

Tell it: "Add the evidence the root post did not have room for: one data point, one caveat, and one next signal to watch."

UGC Radar Agent

Community

Job

Find people creating proof: screenshots, "BTCMind caught this", partner mentions, and analyst use cases.

Watch

  • BTCMind mentions
  • Product screenshots
  • User wins and avoided-loss stories
  • Partner or integration proof

Output

Ranked proof queue for Reposter and Quoter.

Tell it: "Only surface proof that shows BTCMind changed understanding or action. Ignore empty praise and generic giveaway behavior."

Reposter Agent

Action

Job

Amplify credible external proof. This is Nansen's community flywheel.

Accept

  • User proof with BTCMind output
  • Partner announcement
  • Analyst using BTCMind evidence
  • Founder or ecosystem validation

Reject

Empty compliments, low-trust token calls, return promises, and posts that imply financial advice.

Tell it: "Repost only when the post is proof BTCMind is useful. If interpretation is needed, send it to Quoter instead."

Quoter Agent

Action

Job

Quote strong external posts when BTCMind can add the missing mechanism or data lens.

Quote When

  • There is a counter-signal
  • The post misses a hidden constraint
  • BTCMind can add wallet or flow context
  • Operators need a better metric

Output

Short quote take with one mechanism and one implication.

Tell it: "Do not agree generically. Translate the source into what builders, traders, or operators should watch."

Replier Agent

Action

Job

Join conversations under target accounts with a useful mechanism or sharp question.

Shapes

  • "The hidden constraint is..."
  • "The useful split is..."
  • "This only matters if..."
  • "What would change my mind is..."

Output

One or two sentence reply. No generic agreement.

Tell it: "Add one missing mechanism or ask one operator-grade question. If you cannot add either, skip."

Audience Scout Agent

Intelligence

Job

Maintain the target graph: who BTCMind should learn from, reply to, quote, and recruit.

Initial Orbits

  • Onchain analysts
  • Hyperliquid/data traders
  • AI-agent crypto builders
  • Base, Virtuals, OKX, Solana, Injective, Mantle-style ecosystems

Output

Target accounts, community clusters, and priority interaction lists.

Tell it: "Find accounts whose audience overlaps with BTCMind's proof: onchain intelligence, agentic trading, wallet behavior, custody, stablecoins, and market structure."

Narrative Radar Agent

Intelligence

Job

Watch X for narratives BTCMind can explain better than the crowd.

Topics

  • BTC ETF and treasury flows
  • Stablecoin payments
  • Wallet UX and custody failures
  • Crypto AI agents
  • Bridge, security, and market-structure risk

Output

Candidate topics for Signal Source, Quoter, and Replier.

Tell it: "Find high-attention narratives where the crowd is measuring the wrong thing."

Scheduler Agent

Control

Job

Sequence posting, continuation replies, quotes, reposts, and replies without crowding the account.

Starting Windows

  • 2am PT: data original
  • 5am PT: continuation
  • 11am-1pm PT: quote or reply
  • 5pm-8pm PT: proof repost or second original

Output

Daily execution plan and caps per category.

Tell it: "Preserve the proof loop. Do not post an original without planning its continuation or proof capture."

Safety Gate Agent

Safety

Job

Block posts that create reputation, compliance, or trust risk.

Block

  • Price targets and guaranteed returns
  • Thin-liquidity token shills
  • Unverified claims about named people or funds
  • Financial advice language

Require

Evidence, uncertainty, and a next signal to watch.

Tell it: "If the post cannot show evidence or uncertainty, block it. BTCMind should sound useful, not certain."

Analytics Learner Agent

Control

Job

Update templates, timing, and target lists from performance.

Track

  • Engagement by template and time
  • Replies and reposts generated
  • UGC created
  • Inbound leads or product usage

Output

Weekly weights for Poster, Quoter, Replier, and Reposter.

Tell it: "Reward templates that create downstream proof, not just likes."
